# MEDS Alumni Analysis

This repository contains preliminary analyses of workplace outcomes for [Master of Environmental Data Science](https://bren.ucsb.edu/masters-programs/master-environmental-data-science) (MEDS) graduates. 

The internal dataset was sourced from the Bren Career Development Team, and updated using graduates' public LinkedIn profiles. Information on placement titles, organizations, and timelines are included in this data. The MEDS program has been around for 5 years, so data from the Class of 2022 through the Class of 2026 (my year!) is available.

This analysis uses R's {ggplot2} package for visualizations.

Analyses were conducted for the MEDS team at the [Bren School of Environmental Science & Management](https://bren.ucsb.edu/) by Sofia Sarak.
